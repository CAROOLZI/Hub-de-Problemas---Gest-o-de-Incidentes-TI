# Hub-de-Problemas---Gest-o-de-Incidentes-TI
 O Hub de Problemas é uma solução desenvolvida para centralizar e organizar a gestão de tickets e falhas técnicas em ambientes corporativos. ## Levantamento de Requisitos Como parte da análise de requisitos, foram identificadas as seguintes necessidades: - Centralização de falhas de hardware e software. - Classificação por níveis de criticidade
